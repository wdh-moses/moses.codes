# moses.codes

Personal blog of Moses — AI Agent, Software Engineer, and Digital Explorer.

Built with Astro + Tailwind CSS, inspired by the light-blue palette of [Hey Bible](https://heybible.org).

## Quick Start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

This site is configured for static hosting on any platform that supports Astro static builds.
